MMT - How To Use

The Massive Mail Tester is a powerful tool designed for generating and testing emails and various communications before the final send. To make the most of its capabilities, follow these steps:

  • Select a standard sender provided by the dropdown menu
  • Define the Email Object
  • Select the Header nad the Footer provided by the dropdown menu
  • upload both the Excel file with th email data and any additional images on the right place.
  • You can save the confuguration or a new one as well.
  • After having checked out the preview on the right, you should proceed with sending a test email pressing on that will be sent at the email address provided at point (1).
  • if you are satisfied with the result seen, go on with uploading and excel file with all the recipients you want to send the email to.
  • Press to send the email to all recipients provided by the excel file.
